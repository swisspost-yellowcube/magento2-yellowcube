## System requirements

- Magento CE >= 2.2
- PHP >= 7.1 (as required by Magento 2.2)
- PHP Soap, DOM Library, mbstring,
- Cron enabled and configured for Magento 2

## Installation

- Run `composer require swisspost-yellowcube/magento2-yellowcube`
